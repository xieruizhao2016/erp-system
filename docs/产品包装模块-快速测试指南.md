## ✅ 功能测试清单

### 测试1：访问产品包装管理页面
1. 登录ERP系统 (http://localhost:9002)
2. 在左侧菜单栏查找 **"产品包装"** 菜单
3. 点击进入产品包装管理页面
4. ✅ **验证**：页面正常加载，显示空列表或现有数据

### 测试2：新增产品包装
1. 点击页面上方的 **"新增"** 按钮
2. 填写以下信息：
   - **包装编码**：`PKG-001`
   - **包装名称**：`标准纸箱包装`
   - **状态**：启用
   - **装箱数量**：`10` 件/箱
   - **单箱毛重**：`5.5` KG
   - **单箱净重**：`5.0` KG
   - **内箱尺寸**：`30x20x15`
   - **外箱尺寸**：`32x22x17`
   - **包装材料**：`瓦楞纸`
   - **包装类型**：`纸箱`
3. 点击 **"确定"** 保存
4. ✅ **验证**：提示创建成功，列表中出现新记录

### 测试3：编辑产品包装
1. 在列表中找到刚创建的记录
2. 点击 **"编辑"** 按钮
3. 修改 **装箱数量** 为 `20`
4. 点击 **"确定"** 保存
5. ✅ **验证**：提示更新成功，列表中数据已更新

### 测试4：搜索和筛选
1. 在搜索栏输入 **包装编码**：`PKG-001`
2. 点击 **"搜索"** 按钮
3. ✅ **验证**：只显示匹配的记录

4. 点击 **"重置"** 按钮
5. ✅ **验证**：显示所有记录

### 测试5：产品关联包装
1. 进入 **"产品管理"** 页面
2. 点击 **"新增"** 或编辑已有产品
3. 在表单中找到 **"产品包装"** 下拉框
4. 选择刚创建的包装：`PKG-001 - 标准纸箱包装`
5. 填写其他必填字段，点击 **"确定"** 保存
6. ✅ **验证**：产品保存成功，包装信息已关联

### 测试6：Excel导出
1. 返回产品包装管理页面
2. 点击 **"导出"** 按钮
3. ✅ **验证**：下载Excel文件，打开查看数据正确

### 测试7：删除产品包装
1. 在列表中选择一个测试记录
2. 点击 **"删除"** 按钮
3. 在确认对话框中点击 **"确定"**
4. ✅ **验证**：提示删除成功，记录从列表中消失

## 🔍 常见问题排查

### 问题1：菜单中找不到"产品包装"
**解决方案**：
```bash
# 重新执行菜单SQL
cd /Users/xierui/Documents/Project/erp-system
docker exec -i yudao-mysql mysql -uroot -p123456 ruoyi-vue-pro < sql/mysql/erp_product_package_menu.sql
```

### 问题2：API接口404错误
**解决方案**：
1. 确认后端已重启
2. 检查后端日志：
```bash
cd /Users/xierui/Documents/Project/erp-system/yudao-server
tail -f logs/sys-info.log
```

### 问题3：前端页面空白或报错
**解决方案**：
1. 清除浏览器缓存并强制刷新（Cmd+Shift+R）
2. 检查浏览器控制台错误信息
3. 确认前端开发服务器正在运行：
```bash
cd /Users/xierui/Documents/Project/erp-system/original-yudao-ui
npm run dev
```

### 问题4：产品表单中没有"产品包装"选项
**解决方案**：
1. 确认前端代码已更新
2. 清除浏览器缓存
3. 检查ProductForm.vue是否正确修改
