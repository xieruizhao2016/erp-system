# 宝塔面板无法访问 - 排查指南

## 🔍 问题现象

访问 `https://115.190.240.137:23448/a39fc057` 时显示：
- **错误**: ERR_CONNECTION_CLOSED
- **提示**: 115.190.240.137 意外终止了连接

## ✅ 已完成的修复

1. ✅ **防火墙端口已开放**
   - 已在服务器防火墙中开放端口 23448
   - 使用命令：`firewall-cmd --permanent --add-port=23448/tcp`

2. ✅ **宝塔服务已重启**
   - 服务状态正常
   - 端口正在监听

## ⚠️ 需要检查的配置

### 1. 安全组规则（最重要！）

**必须在火山引擎控制台开放端口 23448！**

**步骤：**
1. 登录火山引擎控制台
2. 进入 **云服务器 ECS** → **安全组**
3. 找到你的安全组（Default）
4. 点击 **访问规则** → **入向规则**
5. 点击 **添加规则**

**规则配置：**
```
规则名称: 宝塔面板
协议类型: TCP
端口范围: 23448
源地址: 0.0.0.0/0  (或限制为你的IP，更安全)
动作: 允许
优先级: 1
```

6. 点击 **确定** 保存

### 2. 验证配置

配置完成后，等待1-2分钟，然后：

1. **清除浏览器缓存**
2. **重新访问**: `https://115.190.240.137:23448/a39fc057`
3. **如果提示不安全**，点击"高级" → "继续访问"

## 🔧 其他可能的原因

### 如果仍然无法访问，检查：

1. **检查服务状态**
   ```bash
   ssh erp-server
   /etc/init.d/bt status
   ```

2. **检查端口监听**
   ```bash
   netstat -tlnp | grep 23448
   ```

3. **检查防火墙**
   ```bash
   firewall-cmd --list-ports
   ```

4. **重启宝塔服务**
   ```bash
   /etc/init.d/bt restart
   ```
