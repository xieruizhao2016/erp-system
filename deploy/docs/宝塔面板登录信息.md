### 2. 首次访问安全提示

首次打开面板时，浏览器会提示"不安全"，这是正常的：

**解决方法：**
1. 点击 **"高级"** → **"继续访问"** 或 **"接受风险并继续"**
2. 或者下载证书安装：
   - Windows/Mac证书：https://dg2.bt.cn/ssl/baota_root.pfx
   - 密码：`www.bt.cn`
   - Mac专用证书：https://dg2.bt.cn/ssl/mac.crt

### 3. 安全建议

登录后请立即：
1. **修改面板端口**（可选，但建议）
   - 面板设置 → 面板端口 → 修改为其他端口
   - 记得在安全组中开放新端口

2. **修改用户名和密码**
   - 面板设置 → 修改面板用户 → 修改用户名和密码

3. **绑定IP**（推荐）
   - 面板设置 → 安全设置 → 授权IP
   - 添加你的IP地址，限制只有你的IP可以访问
